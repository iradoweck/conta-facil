# Conta Fácil - Visão Geral do Projeto

**Conta Fácil** é um aplicativo mobile focado em gestão financeira simples e educação fiscal para o mercado de Moçambique, liderado por **Edmilson Adelino Muacigarro**.

## Características Principais
- **Gestão Financeira**: Controle de Entradas/Saídas com cálculo automático de Lucro/Prejuízo.
- **Educação Fiscal**: Alertas e simuladores de impostos para pequenos negócios.
- **Chat Inteligente**: Assistente virtual (Bot) com escalonamento para atendimento humano (WhatsApp).
- **Marca Pessoal**: Integração com os serviços profissionais de Edmilson Muacigarro.

## Tecnologias
- **Mobile**: Flutter
- **Backend**: Firebase / Laravel API
- **Banco de Dados**: Firestore / PostgreSQL
- **Autenticação**: Firebase Auth + Google Login

## Estrutura de Atendimento
- **Nível 1**: Bot (Baseado em Conhecimento Específico)
- **Nível 2**: Humano (Edmilson) via WhatsApp / Chat Interno
