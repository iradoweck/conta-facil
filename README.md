# 💰 Conta Fácil

![Branding](assets/images/logo.png)

> **Controle simples. Negócios fortes.**

O **Conta Fácil** é o parceiro financeiro definitivo para o empreendedor moçambicano. Mais do que um simples gestor de despesas, é uma ferramenta estratégica que une contabilidade rigorosa, educação fiscal e o acompanhamento próximo de um **Parceiro AI** baseado na experiência de **Edmilson Muacigarro**.

---

## 🌟 Características Únicas

| Funcionalidade | Descrição |
| :--- | :--- |
| 🏦 **Gestão Multi-Contas** | Separe suas finanças de Negócio e Pessoal em contas distintas (Banco, M-Pesa, Carteira). |
| 📈 **Relatórios Reais** | DRE, Fluxo de Caixa e Balanço Patrimonial gerados automaticamente. |
| 🤖 **Parceiro AI** | Um assistente que fala a sua língua e te motiva a atingir suas metas. |
| 🇲🇿 **Educação Fiscal** | Simuladores integrados para ISPC e conformidade com as regras de Moçambique. |
| 💼 **Perfil Edmilson** | Acesso direto aos serviços profissionais de assessoria contábil e desenvolvimento. |

---

## 📜 Nossos Dogmas e Protocolos

Para garantir a excelência do **Conta Fácil**, seguimos estes princípios inegociáveis:
1.  **DOGMA DO LOCALHOST**: O app deve estar sempre rodando (`flutter run`) durante o desenvolvimento.
2.  **FEEDBACK IMEDIATO**: Mudanças de código devem ser refletidas instantaneamente para auditoria visual.
3.  **ESTABILIDADE CORE**: Funcionalidade real vem antes de estética vazia. Dados não podem ser perdidos.

## 🚀 Tecnologias

- **Frontend**: Flutter (Material 3)
- **Estado**: Riverpod (Gestão Reativa)
- **Persistência**: SharedPreferences (Local Storage Seguro)
- **Backend**: Firebase Authentication
- **Design**: Outfit & Inter (Google Fonts)

## 🛠 Como Iniciar

1. Certifique-se de ter o Flutter instalado.
2. Execute o comando para instalar dependências:
   ```bash
   flutter pub get
   ```
3. Inicie o app no Chrome (obedecendo ao Dogma do Localhost):
   ```bash
   flutter run -d chrome --web-port 5000
   ```
